# rakuten-ll
Training in Cebu.
## rakuten-guidance-api
Necessary package

- flask
- pymysql

### Start API (flask)

```
cd rakuten-ll/rakuten-guidance-api
python app.py
```

### Access API
If you start in localhost, in your browser you write following http request.
```
http://127.0.0.1:5000/teachers/search/?hoge=fuga 
```
